# Поздравлятор

**Поздравлятор** - Данное приложение предназначено для ведения списка друзей/знакомых/сотрудников, у которых в ближайшее время было или будет День Рождения.

## Основные возможности

- Отображение списка ближайших пользователей, у которых будет (или был) День Рождения
- Добавление нового пользователя с указанием даты рождения и фотографии
- Отображение списка всех пользователей, добавленных в приложение
- Редактирование данных пользователя из списка
- Удаление пользователя из списка
- Хранение и отображение фотографий именинников.
